Synchronization API

API name

Input

Output

Description

sensorQtyAdjust (sensorNumber, jsonConfig)

int sensorNumber: number of sensors would be add to the synchronization process.

json jsonConfig: current configuration.

The result of sensors adjustment execution.

This function is used to adjust the number of sensors that the system has to handle.

deadlineAdjust (dueDate, jsonConfig)

int dueDate: synchronization window.

json jsonConfig: current configuration.

The result of deadline adjustment execution.

This function is used to adjust the deadline of synchronization window.

destAdjust (jsonUrl, jsonConfig)

json jsonUrl: list of url destinations in json format that the synced data would be sent to.

json jsonConfig: current configuration.

The result of destination adjustment execution.

This function is used to adjust the list of url destinations. The synced data would be sent to every member of this list.

coreSync (objLoki, mUserId, history, nDataSource, jsonConfig, badMerge, historyRecord, syncType)

lokijs objLoki: the reference to the buffer collection.

int mUserId: the id of the user.

lokijs history: the reference to the history collection.

int nDataSource: number of current sensors.

json jsonConfig: current configuration.

int badMerge: flag for syncing data when exceeding the deadline.

associated_array historyRecord: the record that is used to cover the missing data.

int syncType: type of sync.

 

The result of synchronization execution.

This function is used sync the data.

serverPortAdjust (port, jsonConfig)

int port: the port number to start the synchronization service.

json jsonConfig: current configuration.

The result of port adjustment execution.

This function is used to adjust the port for starting the synchronization service.

bolean isJson (item)

string item: the item that needs to check for the json format.

True/false.

This function is used to verify a string is json or not.

bolean isURL (str)

string str: the item that needs to check for the URL validation.

True/false.

This function is used to verify a string is URL or not.

syncStart (mergeDataBody, jsonConfig)

associated_array mergeDataBody: synced data.

json jsonConfig: current configuration.

The result of sending synced data.

This function is used to send the synced data to the list of predefined destinations (URLs).
